KONTROL YAPILARI



Kontrol yapıları Programlama dünyasının olmazsa olmazlarından kontrol yapıları, programın akışının kontrol edilmesini sağlar. Örneğin; Facebook, Twitter gibi büyük sitelere dahi girerken kullanıcı adı ve şifrenizin doğruluğunu kontrol yapıları ile kontrol ederler ve buna göre işlem yaparlar.



IF yapısı

Programlama dünyasında en çok kullanılan yapıdır. Neredeyse bütün programlama dillerinde kulanım şekli benzerdir. Kullanımına bakacak olursak:

#include

int main()
{
if(koşul) {
eğer koşul sağlanıyorsa burası çalışır;
              }
}

Test edilen ifade yani koşulumuz sağlanıyorsa IF yapısı içindeki kodlar çalışır; koşulumuz sağlanmıyorsa IF yapısı içindeki kodlar işlemeden atlanır.






IF ELSE Yapısı

İlk başlıkta koşulun sağlandığı durumdan söz ettik. Peki ya koşul sağlanmıyorsa ne olacak? Bu sefer işin içine ELSE devreye giriyor. Yani IF yapısında belirtilen koşul sağlanmıyorsa işlenecek olan kodları ELSE ifadesinde belirtiriz.Kullanımı basitçe şu şekildedir:

#include

int main()
if ( koşul )
{
koşul sağlanıyorsa çalışacak kod;

}else

...
